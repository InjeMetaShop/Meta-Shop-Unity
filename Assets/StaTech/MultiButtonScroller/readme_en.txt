Multi Button Scroller is Asset which supports to make scrollable UI button elements.
You can use this asset on several genre games like social game and casual game.
It's easy to embed your project because Multi Button Scroller's main logic and view and model are completely separated.

**Requires DOTween or DOTween Pro to be imported and setup in the project before importing the Multi Button Scroller package.**

#How to use
*This asset needs to install DoTween.
Please find tutorial.pdf
